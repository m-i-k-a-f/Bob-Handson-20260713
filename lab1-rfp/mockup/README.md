# 工事原価管理システム　操作ご案内

**ワトソン建設株式会社　ご担当者様へ**

このファイルは、工事原価管理システム刷新プロジェクトの**画面イメージ（モックアップ）**をご確認いただくための資料です。
システムの開発・インストールは一切不要です。ファイルをダブルクリックするだけでご確認いただけます。

---

## 開き方

**手順は1ステップだけです。**

1. `index.html` というファイルを**ダブルクリック**してください。

お使いのパソコンのブラウザ（Chrome、Edge、Safari など）が自動的に起動し、画面が表示されます。

> **うまく開かない場合：**
> `index.html` を右クリック →「プログラムから開く」→「Google Chrome」または「Microsoft Edge」を選択してください。

## 画面の見かた

### 開くと最初に見える画面（工事一覧）

画面を開くと、**施工中の3工事が「要注意の工事が上」の順番**で表示されます。

```
┌─────────────────────────────────────────────┐
│ ⚠ 要注意工事 1件：原価率が閾値を超過しています │  ← 赤い帯
└─────────────────────────────────────────────┘

[施工中: 3件]  [受注金額合計]  [平均原価率]  [アラート: 1件]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🔴 要注意   K-001 大阪南港 倉庫新築工事    ← 赤いカード（上に来る）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🟡 注意     K-002 京都駅前 ビル改修工事
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🟢 正常     K-003 名古屋高速 橋梁補修工事
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

#### 色の意味

| 色 | ステータス | 意味 |
|---|---|---|
| 🔴 赤 | **要注意** | 原価率が受注時の想定を **10ポイント以上** 上回っています |
| 🟡 橙 | **注意** | 原価率が受注時の想定を **5〜9ポイント** 上回っています |
| 🟢 緑 | **正常** | 原価率が受注時の想定の範囲内に収まっています |

---

### 工種別の内訳を見る

工事カードを**クリック**すると、その工事の**工種ごとの詳細**が展開されます。

- **実行予算**と**発生原価**の比較が工種別に確認できます
- 予算を超えている工種は赤、予算内に収まっている工種は緑で表示されます
- グラフでどの工種がどれだけずれているかを視覚的に確認できます

もう一度クリックすると詳細が閉じます。

## 画面の切り替えかた

画面上部に3つのタブがあります。クリックで切り替えできます。

```
[工事一覧 🔴1]  [工種別内訳]  [協力会社別]
```

### タブ①　工事一覧

**→ 「どの工事が危ないか」を一覧で確認する画面です。**

全工事の採算状況をひと目で把握できます。

---

### タブ②　工種別内訳

**→ 「特定の工事の、どの工種でコストが超過しているか」を確認する画面です。**

画面上部のプルダウン（▼）から確認したい工事を選ぶと、工種ごとの実行予算・発生原価・差異が表形式で表示されます。

**操作手順：**
1. 「表示工事」のプルダウンをクリック
2. 確認したい工事名を選択
3. 工種別の表とグラフが自動的に更新されます

---

### タブ③　協力会社別

**→ 「どの協力会社に、いくら発注・支払いをしているか」を確認する画面です。**

工事と工種で絞り込みができます。

**操作手順：**
1. 「工事」プルダウンで対象工事を選択（全工事 or 個別選択）
2. 「工種」プルダウンで対象工種を絞り込み（任意）
3. 表が自動的に更新されます

各行の右端に**支払率バー**が表示されます。バーが短い（支払率が低い）場合は支払いが滞っている可能性があります。

## よくあるご質問

**Q. インターネットに接続していないと見られませんか？**
画面の表示にはインターネット接続は不要です。ただし、フォント（文字のデザイン）の読み込みに接続が必要なため、オフライン環境では文字のデザインが若干変わることがあります。表示内容には影響ありません。

**Q. スマートフォンやタブレットで見られますか？**
このモックアップはパソコンでの表示を前提としています。スマートフォン・タブレット対応については、プロジェクトの次フェーズでご相談させてください。

**Q. データを編集できますか？**
このファイルはあくまで「見た目・操作感のご確認用」です。データの編集機能はありません。

**Q. 画面の数値はどこから来ていますか？**
ファイル内に埋め込まれたサンプルデータです。実際のシステムでは、貴社の会計・購買システムからデータを取得して表示します。
