## 分析上の制約・注記

| 制約 | 内容 |
|---|---|
| **文字コード** | `資材搬入記録_5月.csv` は Shift-JIS で保存。Pythonの自動判定（cp932）により読み込み・突合を実施。 |
| **P-0001・P-0002の工種別内訳** | 実行予算対比Excelに工種別明細がなく、どの工種で超過しているか特定不可能。工種別内訳の整備が急務。 |
| **完工時損失推計の精度** | 進捗率は工事番号一覧CSVの自己申告値。実態と乖離している場合、推計損失額が変動する。 |
| **単価マスタ乖離の年換算** | 5月1か月分の発注実績ベースで計算。月によって発注量は異なるため、年間値は参考値。 |
| **Salesforce↔見積書差異** | 営業会議メモから「値引き要請後の着地額不明」と判断したが、別途確認版見積書が存在する可能性は排除できない。 |
